# 📝 AI 小红书文案生成器

> 基于通义千问大模型的专业级小红书爆款文案创作工具，输入关键词即可一键生成高互动文案。

## 🎯 产品定位

小红书创作者面临日更压力大、平台风格难把控、标题标签试错成本高等痛点。本工具将**爆款方法论 + 大模型能力**结合，让创作者从"想半天写不出"变成"一键生成，微调即发"。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🏷️ 智能标题生成 | 3个备选标题，标注所用公式（数字+结果/反差+悬念/身份+痛点/情绪+共鸣） |
| 📝 爆款正文创作 | 开头3秒抓注意力，中间高信息密度，结尾引导互动 |
| 💬 互动引导语 | 自动生成自然不生硬的点赞/收藏/评论引导 |
| 🏷️ 标签策略 | 8-10个标签，大流量词+长尾精准词分层 |
| 📊 爆款预测 | 预估互动类型、最佳发布时间、封面配图建议 |
| ⭐ 文案评分 | 5维度专业评分（标题吸引力/信息密度/情绪价值/互动潜力/平台适配度） |

## 🎨 创作参数

- **6种文案风格**：种草推荐 / 产品测评 / 教程攻略 / 日常分享 / 避雷吐槽 / 好物合集
- **5种语气调性**：活泼俏皮 / 专业理性 / 温柔治愈 / 犀利直白 / 幽默搞笑
- **4种目标受众**：18-25岁学生党 / 25-30岁职场新人 / 30-35岁精致妈妈 / 泛人群
- **Emoji密度**：5级可调（纯文字 → 每句都有）
- **文案长度**：短(100-200字) / 中(200-400字) / 长(400-600字)

## 🖥️ 界面预览

- 双栏布局：左侧输入创作需求，右侧实时展示生成结果
- 小红书品牌色渐变主题
- 侧边栏历史记录（最近5条）
- 支持 txt / Markdown 双格式导出

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/Jamal-42/xiaohongshu-ai-writer.git
cd xiaohongshu-ai-writer
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 获取 API Key

前往 [阿里云百炼平台](https://bailian.console.aliyun.com/) 注册账号并开通通义千问服务，获取 API Key（有免费额度）。

### 4. 运行应用

```bash
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501`，在左侧配置中心输入 API Key 即可开始创作。

## 🧠 Prompt 工程亮点

本项目的核心竞争力在于 Prompt 设计，经过3轮迭代优化：

| 版本 | 策略 | 效果 |
|------|------|------|
| V1 | 基础指令式 | 能生成但质量不稳定，字数常超标 |
| V2 | 增加硬约束+读者画像 | 字数可控，风格区分度提升 |
| V3 | Few-shot示例+标题公式+写作铁律+受众分层 | 输出接近真实爆款水平 |

**关键设计思路：**
- 角色设定精确化（"年收入百万的头部博主"而非泛泛的"文案创作者"）
- Few-shot 注入真实高赞案例，让模型学习结构模式
- 标题公式体系化，强制标注所用公式，输出可复用
- 分层标签策略（大词引流 + 长尾精准）

## 📌 使用建议

1. **关键词越具体越好**："油皮学生党 50元内防晒霜 不搓泥" 远优于 "防晒"
2. **善用补充信息**：填入价格区间、适用人群、想突出的卖点
3. **先生成再评分**：用评分功能获取改进建议，针对性优化
4. **多试几种风格**：同一个主题换不同风格/语气可能有惊喜效果
